Changelog & Release Notes

Follow the ongoing evolution of PixelResize. We are continuously improving our image compression algorithms, adding new formats, and optimizing browser performance.

v2.0.0 Current Release

Released on July 4, 2026

  • 🚀 Performance Improvements: Enhanced local rendering loops and canvas instantiation logic. Reduced image loading latency for massive 8K image files.
  • 📈 SEO Updates: Comprehensive structured data (JSON-LD) implementation supporting BreadcrumbList, WebSite with SearchAction, SoftwareApplication, and VideoObject.
  • 🛠️ New Tools: Added Discord Profile Picture (PFP) Resizer, Instagram Aspect Ratio Resizer, Passport Photo Size Resizer, and HEIC to JPG converter natively.
  • 🐞 Bug Fixes: Fixed transient memory leaks on mobile viewports when performing large bulk file exports. Improved alignment behavior for tool category icons.
